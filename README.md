# Sales-Dashboard
This project is a simple dynamic dashboard that is used for an analysis of the Sales of different type of cars/vehicles from 2003-2005.
## Skills used
- Data cleaning using Power Pivot
- Formulas (If function, Text/data function)
- Pivot table 
- Pivot charts
- Slicers
- Sheet locking/slicer properties
- Geographical Map of data
- Charts (pie chart and line chart)
- Illustrations

## There are 4 parts of the excel file
- First is for the dynamic dashboard itself where you can see the summaries which you can filter out the year and type of products through slicers
- Second is for the pivot charts that I made for doing the dashboard
- Third is for the cleaned dataset that I cleaned using powerpivot (except for phone numbers)
- Fourth is the raw dataset that I got from kaggle sample dataset
